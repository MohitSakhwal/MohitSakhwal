## 🚀 Featured Projects

### 🤖 PolicyPulse — RAG-Based Compliance Q&A System
> *LangChain · OpenAI GPT · ChromaDB/Pinecone · RBAC*

Built an end-to-end **Retrieval-Augmented Generation** pipeline that answers employee queries over 500+ pages of HR handbooks and legal compliance manuals. Implemented semantic chunking, metadata filtering, and role-based access control at the retrieval layer.

**Key results:** Reduced hallucination through prompt tuning; context-aware retrieval across dense, fragmented policy docs.

---

### 📉 Customer Churn Prediction & Segmentation *(Telco Dataset)*
> *Python · Scikit-learn · SQL · Power BI*

Analysed 7,000+ customer records to surface top churn drivers (contract type, tenure, monthly charges). Built a **Logistic Regression classifier (82% accuracy)** and segmented customers into High / Mid / Low risk tiers for targeted retention.

**Key results:** Estimated 15% churn reduction potential; churn scores exported directly into Power BI dashboard.

---

### 📊 Superstore Sales — Multi-Page Power BI Dashboard
> *Power BI · DAX · Star Schema · Data Modelling*

Built a production-grade multi-page Power BI dashboard with a full star schema, custom DAX measures, and time-intelligence calculations. Documented the project with a full GitHub README including dataset stats, data model, and business insights.

---

### 🏠 House Prices — End-to-End ML Pipeline *(Kaggle)*
> *Python · XGBoost · LightGBM · SHAP · joblib*

Complete EDA → feature engineering → multi-model comparison → SHAP explainability → joblib pipeline export. Focused on interpretable ML to understand what drives housing valuations.
